## 🚀 البدء السريع

<br/>

### 1️⃣ استنسخ المشروع

```bash
git clone https://github.com/yourusername/nibras.git
cd nibras
```

### 2️⃣ ثبّت المكتبات

```bash
npm install
```

### 3️⃣ شغّل بيئة التطوير

```bash
npm run dev
# افتح المتصفح على: http://localhost:5173
```

### 4️⃣ بناء نسخة الإنتاج

```bash
npm run build
# الناتج في مجلد: dist/
```

## 🗄️ إعداد قاعدة البيانات (Supabase)

<br/>

> ⚠️ **مهم:** يجب تشغيل هذه الخطوة قبل استخدام التطبيق لأول مرة.

<br/>

1. افتح مشروعك على [supabase.com](https://supabase.com)
2. اذهب إلى **SQL Editor** ← **New Query**
3. انسخ محتوى ملف `supabase-setup.sql` والصقه
4. اضغط **Run**

**سيتم إنشاء الجداول التالية تلقائياً:**

```sql
profiles        -- ملفات المستخدمين (مرتبطة بـ auth.users)
files           -- الملفات المرفوعة
chat_sessions   -- جلسات المحادثة
quiz_sessions   -- جلسات الاختبارات
exams           -- الامتحانات والمواعيد
study_plan      -- خطة الدراسة
```

## 🤖 إعداد الذكاء الاصطناعي

<br/>

### نماذج مجانية موصى بها (OpenRouter)

| النموذج | السرعة | الجودة | مناسب لـ |
|---------|--------|--------|---------|
| `meta-llama/llama-3.1-8b-instruct:free` | ⚡⚡⚡ | ⭐⭐⭐ | المحادثة اليومية |
| `google/gemma-2-9b-it:free` | ⚡⚡ | ⭐⭐⭐⭐ | الشرح والتلخيص |
| `deepseek/deepseek-r1:free` | ⚡ | ⭐⭐⭐⭐⭐ | التفكير المعقد |
| `qwen/qwen-2.5-7b-instruct:free` | ⚡⚡⚡ | ⭐⭐⭐ | متعدد اللغات |

<br/>

**للبدء:**
1. سجّل في [openrouter.ai](https://openrouter.ai) — مجاني
2. احصل على مفتاح API من صفحة **Keys**
3. افتح إعدادات نِبْرَاس ← ضع مفتاحك ← اختر النموذج ← احفظ ✅
